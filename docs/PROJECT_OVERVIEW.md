# Kelola J.Family

Kelola J.Family adalah aplikasi manajemen keuangan keluarga yang memiliki berbagai bisnis.

Tujuan aplikasi:
- Mengelola pemasukan dan pengeluaran keluarga
- Memantau profit tiap bisnis keluarga
- Mengontrol pengeluaran rumah tangga
- Membantu keluarga mencapai target finansial

Target pengguna:
2-6 anggota keluarga dalam satu family group.

Platform:
- Web (Next.js)
- Mobile friendly
